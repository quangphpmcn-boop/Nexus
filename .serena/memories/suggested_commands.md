# Suggested Commands

## System (Windows)
- `git` тАФ version control
- `node` / `npm` тАФ Node.js runtime (for projects using it)
- `powershell` тАФ shell, script execution

## Nexus CLI
- `nexus init` тАФ Initialize Nexus in a project
- `nexus install` тАФ Install framework files
- Run via: `H:\Kit\Nexus\bin\nexus.ps1`

## Development
- `git status` тАФ check working tree
- `git diff` тАФ review changes
- `git add -A && git commit -m "..."` тАФ atomic commits (conventional format)

## Testing
- No automated test suite тАФ framework is markdown/config-based
- Verification via `/verify` and `/health` workflows
